### URL
```groovy
https://maps.mapmyindia.com/{search-keywords}/near/xx.xxxxx,yy.yyyyy
```
<br>


### URL Parameters
1. `search-keywords`: Nearby search keywords
2. `xx.xxxxx,yy.yyyyy`: Lat/Long pair
